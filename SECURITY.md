# Security Policy

## Supported Versions

Only the latest release version receives active security patches:

| Version | Supported |
| :--- | :--- |
| v0.1.x | Yes (Active) |
| < v0.1.0 | No |

## Reporting a Vulnerability

**DO NOT create public GitHub issues for security vulnerabilities.**

If you discover a security vulnerability, please report it via email to **security@nomen.ai**. 

Include the following details in your report:
- Name and description of the vulnerability.
- Steps to reproduce or proof-of-concept script.
- Potential impact.

We will acknowledge receipt of your report within 48 hours and coordinate a patch release following responsible disclosure guidelines.
